About pick this

pick this was built by Ben Bougher, Jacob Foshee, Evan Bianco, and a combination of Chris Chalcraft and Matt Hall at the 2014 Geophysics Hackathon in Denver, Colorado. The demo version we built at that event had a hard-coded image and no voting, but the basic concept was all there. Jacob is the Javascript mastermind.

 

Support this project!

We want pick this to be free to use. But we have a long list of enhancements to add and a short list of bugs to fix. If you would like to donate time or money to the project, please get in touch. The top 3 supporters will earn a place on the site, on everything connected with the project, and (best of all) in our hearts.

 

The geeky details

pick this is open source software. You can find it on GitHub. The software is written in Python and JavaScript, and uses the Jinja2 templating engine and the Bootstrap HTML/CSS/JS framework. The drawing is handled by Raphaël. The web app you are using is running on Google Cloud Platform.
